Jak działają ustawienia
• Provider do generowania — wybór „tylko X” oznacza, że aplikacja nie próbuje innych API (np. wyłączonego LM Studio).
• Wartości wpisane tutaj nadpisują wartości z pliku .env dla Twojego konta.
• Jeśli pole jest puste, aplikacja używa wartości z .env.
• Klucze API są przechowywane w lokalnej bazie SQLite — nie są wysyłane nigdzie indziej.
• Testuj provider — wysyła żądanie testowe bezpośrednio z aktualnie wpisanych wartości (bez zapisywania).
• Jeśli test pokazuje błąd: sprawdź saldo konta, poprawność klucza i czy model istnieje.
